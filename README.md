# Awesome Auto Research Hub

A curated hub for AI systems that automate scientific research, including literature review, survey synthesis, experiment design and execution, paper writing, AI scientists, and autonomous research agents.

This repository is generated by [awesome-hub-generator](https://github.com/jin-s13/awesome-hub-generator).

## Contents

- `awesome.yaml` - hub configuration and upstream sources.
- `data/` - generated papers, datasets, projects, taxonomy, and survey metadata.
- `website/` - Astro frontend for browsing the hub.
- `website/public/assets/papers/` - localized teaser images used by the frontend.

## Local Preview

```bash
cd website
npm install
npm run dev -- --host 127.0.0.1 --port 4322
```

Then open:

```text
http://127.0.0.1:4322/awesome-auto-research-hub/zh/
```
