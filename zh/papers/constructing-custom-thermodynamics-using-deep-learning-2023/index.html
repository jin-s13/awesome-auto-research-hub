深度分析

创新点

  • 基于广义Onsager原理的平台，直接从微观轨迹学习宏观动力学描述
  • 同时构建约化热力学坐标并解释这些坐标上的动力学
  • 结合AI与已知物理原理（如对称性和守恒定律）的自动假设生成与验证

方法

该方法使用广义Onsager原理，从微观轨迹观测中学习随机耗散系统的宏观动力学。它同时构建约化热力学坐标并解释动力学，在外部场中的聚合物拉伸上得到验证。

关键结果

该方法学习了聚合物拉伸的三个可解释热力学坐标，构建了识别稳定态和过渡态的动力学景观，并实现了对拉伸速率的控制，经实验验证。
