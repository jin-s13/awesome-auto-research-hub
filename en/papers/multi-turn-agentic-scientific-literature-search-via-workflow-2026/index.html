Deep Analysis

Innovations

  • Framing scientific literature search as workflow induction, constructing an executable DAG of paper-search operators
  • Explicit, editable search workflows that can be refined by user feedback, enabling control and inspectability
  • Training via supervised workflow imitation and preference optimization over controlled workflow corruptions
  • Introduction of PaperPilot, a multi-turn agentic search agent that uses workflow induction

Methodology

PaperPilot takes an anchor paper and a user query, builds an executable DAG of operators (keyword search, citation expansion, filtering, scoring, reranking, evidence extraction), and refines both query and workflow via user feedback. It is trained with supervised imitation of workflows and preference optimization on corrupted workflows.

Key Results

PaperPilot-9B improves over the base Qwen3.5-9B toolset agent, with Hit@5 from 58.0 to 77.0, MRR from 47.5 to 59.4, nDCG@10 from 26.8 to 32.5, and workflow execution errors reduced from 9.5% to 0%.
